5. Attempted fisheye distortion (learned CSS limitations, removed for clean approach)
6. Final bubble styling: 1px white border with pink/blue glow, radial highlight, 98% photo opacity

### Technical Lessons Learned
- CSS `transform: scale()` doesn't create true fisheye - just enlarges
- True distortion requires SVG filters, Canvas, or WebGL
- Simple is often better - clean glassy effect beats complex distortion
- Layer-based debugging effective for nested scroll issues
- `lg:hidden` class useful for mobile-only features

### User Feedback Patterns
- Iterative refinement: "way more fisheyed" ‚Üí multiple attempts ‚Üí "let's remove the attempt"
- Clear preferences: "i liked it before better, just not so intense"
- Specific observations: "the 3 you added are HUGE"
- Direct corrections: "you are changing the desktop. you never should have..."
